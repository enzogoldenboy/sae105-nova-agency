# Learning Website Project

This project was created as part of my journey to learn and practice web development using **HTML**, **CSS**, and **JavaScript**.

## 📖 About

The goal of this project was to understand the fundamentals of front-end development, including:

- Structuring web pages with HTML
- Styling layouts and components with CSS
- Adding interactivity with JavaScript
- Organizing files in a simple web project

## 🚀 Features

- Responsive layout
- Modern UI design
- Interactive elements with JavaScript
- Clean and beginner-friendly code structure

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla JS)
